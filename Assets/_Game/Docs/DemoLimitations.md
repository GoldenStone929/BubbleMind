# Demo Limitations

- 这是离线玩法原型，不是生产游戏或联网服务。
- 存档、货币和抽卡均在本机执行，不具备服务器权威性或防作弊能力。
- 当前阵容包含水、火、土、风、雷五套 Blender 基础史莱姆和一名限定 UR 黑洞史莱姆；六名标准角色会复用五套基础模型，当前元素仅代表视觉身份，仍不是最终角色差异化阵容。
- 当前只有单抽；十连、保底、重复碎片、角色升级和完整养成属于 P1。
- 没有登录、网络、IAP、广告、实时活动、远程内容、线上 PvP 或音频。
- 当前移动使用固定 Tick 的直线接近，没有 NavMesh、碰撞绕行或复杂障碍寻路；Tank / Assassin 统一射程 1，其余职业统一射程 5，尚未提供每个关卡的动态射程修正。
- 当前怒气固定为 0–1000、普攻命中 +100、受伤 +50；技能槽 2 / 3 使用 5 秒错峰的固定 10 秒自动时间表，尚未实现手动大招、沉默、打断、施法队列或复杂技能 AI。
- 战斗表现以清晰可读为目标；多人近战和黑洞聚拢时仍可能发生模型或姓名牌相交，部分动作和事件重播可在 P1 继续打磨同步感。
- 基础史莱姆目前使用无骨骼 Transform 动作与元素装饰动画；尚未制作面部表情状态、骨骼蒙皮或每名标准角色的独立动画集。
- Catherine 已使用 19,000 面不对称凝胶模型、纯黑核心与三点布光，但仍是内部原型；正式商业角色还需要完整 Animator、移动端透明过绘优化和参考图商业权利确认。
- 鼠标、触摸、键盘和基础手柄选择可用，但尚未制作完整的无障碍与按键设置页面。
- Windows Build 与 Unity Editor 版本共享相同的本地 Demo 逻辑。
